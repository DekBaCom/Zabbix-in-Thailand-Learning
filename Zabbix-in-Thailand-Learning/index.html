เว็บไซต์นี้ทำงานบน GitHub Pages แบบ Static Site หากต้องการเพิ่มวิดีโอใหม่ สามารถทำได้โดยการแก้ไขไฟล์ data.js ใน Repository นี้:

  1. เปิดไฟล์ data.js
  2. เพิ่มข้อมูลใน Array videoData ตามโครงสร้างเดิม
  3. Commit และ Push การเปลี่ยนแปลง
